Actor Model

The Actor Model provides a higher level of abstraction for writing concurrent and distributed systems. It alleviates the developer from having to deal with explicit locking and thread management, making it easier to write correct concurrent and parallel systems.

Stream Processing

Proto.Actor is designed to optimize for locality, grains can gradually migrate to cluster members from where it receives the most interactions. This allows Proto.Actor to leverage in-process performance for realtime stream processing.

Digital Twins

A digital twin is a virtual representation that serves as the real-time digital counterpart of a physical object or process. By leveraging Proto.Actors local affinity strategy, digital twins can react on external events in realtime.
